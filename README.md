# congb19-top

congb19-top 的前端

## 技术栈

react, scss, docsify。

## 地址

- 本站：http://me.congb19.top
- 博客：https://www.congb19.top
- 文档：http://docs.congb19.top

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### 开始开发

`npm run start`

### 打包

`npm run build`

### 开始文档编写

`npm run docs`

文档站由 github pages 驱动，更新代码即可，无需编译。
