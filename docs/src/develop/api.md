## Public

> 获取日期

- url

`GET` /public/getDate

- Parameters

```javascript
{
  "isLogin": true //是否登录
}
```

- Responses

  - Code: 200

  - Description: Get the current date

  - Example Value:

```javascript
{
  "title": "date",
  "code": 200,
  "data": {
    "year": 2020,
    "month": 11,
    "date": 3,
    "day": 2
  }
}
```
