## 1 基本实习情况

### 1.1 实习项目与实习单位概况

#### 1.1.1 实习项目简介

本人参加的实习项目是公司的主要产品线之一——EduSoho在线网校系统企培版的新模块-知识库的开发。

母产品简介：
Edusoho企培版——为企业快速发展赋能。官网介绍：https://www.edusoho.com/（主要）、https://ct.edusoho.com/（企培版）。

本产品背景：
Situation：
企业中随着业务的发展，会产生大量的知识、经验文档等希望可以存放并供员工使用和学习，员工需要可以即查即用。尤其是涉及大量的业务知识的文档，将这些业务知识快速的同步给一线员工是这些企业的刚性需求，帮助员工解决工作中遇到的业务问题。
Problems：
企业的知识、经验文档没有统一的有序的存放持续的沉淀，员工需要使用内容的时候，很难找到，散落在各处。内容更新不及时，版本混乱。
EduSoho企培版中目前针对企业内部的知识以及经验的内容主要以线上课程的形似来沉淀和呈现，更适合用于体系化的内容学习，在知识库场景下不能很好满足员工即查即看即用的需求。另外从课程的内容呈现形式，课程的内容学习列表，不能满足知识管理和存放以及呈现上的需求。

#### 1.1.2 实习单位与实习指导老师

  本人实习的单位是杭州阔知网络科技有限公司，部门为研发中心-主程序团队。阔知是一家一站式在线教育解决方案服务商，致力于为企业、个人和教育机构提供包括平台、运营、内容和营销在内的在线教育整体解决方案。阔知旗下拥有EduSoho网校、EduSoho教育云、EduSoho企培、EduSoho智慧课堂等产品，截止2019年1月，已服务国内外近10万家企业和教育机构，200多家高等院校，覆盖至美、加、日、英、德等多个发达国家。
  实习的校内指导老师为汤颖，企业指导老师为洪永春，为该公司的前端开发工程师。

2.2.3 产品技术方案

前后端分离
	知识库技术实现上采用前后端分离的架构，前后端分离已成为互联网项目开发的业界标准使用方式。
	在前后端分离架构中，后端只需要负责按照约定的数据格式向前端提供可调用的API服务即可，前端工程师只需要编写HTML、js、CSS等前端资源，然后通 过HTTP请求调用后端提供的服务即可。前后端之间通过HTTP请求进行交互，前端获取到数据后，进行页面的组装和渲染，最终返回给浏览器。
	前后端代码库同样也分离，前端代码中有可以进行Mock测试(通过构造虚拟测试对 象以简化测试环境的方法)的伪后端，能支持前端的独立开发和测试。而后端 代码中除了功能实现外，还有着详细的测试用例，以保证API的可用性，降低 集成风险。

前端技术方案
	Vue.js：Vue 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。
图
	Axios：由于采用前后端分离的架构，因此使用Axios 这一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。
	Ant Design Vue：采用Ant Design UI组件库，ui组件丰富齐全，能满足项目中大部分需求，开发效率高。
图
	Ckeditor4-vue：功能强大，编辑能力强。
